那么现在如何快速部署到Github呢？首先需要安装部署工具

所有站点设置都在根目录_config.yml这个文件里面，然后将其中deploy项配置成如下内容，其中xxx是自己的Github昵称

deploy:
type: git
repository: git@github.com:xxx/xxx.github.io.git
branch: master
